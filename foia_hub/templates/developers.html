Technology

FOIA Hub is a Python/Django application. Many of the scripts written to collect data are also written in Python.

API

All of the information presented is available through an API.
  • API documentation

Bulk Data

The data source for the API and the tool is a set of YAML files that contain all the data. These files are the canonical source for our data and are updated through a combination of automated and manual processes.

  • YAML data files

Repositories

  • foia Our main repository (also contains the bulk data)
  • foia-hub The repository for FOIA Hub.

Data Sources

We use a number of data sources for the information we display about each component of the government.

Our contact and processing times data comes from a combination of foia.gov sources and manually determined data. Descriptions and acronyms come from a usa.gov API. We populate they keyword search terms using the Federal Register API. Finally, the FOIA library links are scraped from each agency's FOIA web page where relevant.

Our source code provides more information and details on how each data source is processed.
