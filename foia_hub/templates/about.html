About this Project

openFOIA aims to improve the FOIA request experience.

openFOIA is an online tool that helps users find current FOIA contact and request information across the federal government. It is a work in progress by the Department of Justice and 18F.

Our work is one of several initiatives currently in progress to modernize the Freedom of Information Act, as part of the United States’ Second Open Government Partnership National Action Plan. openFOIA supplements the resources available on FOIA.gov, the government’s central website on FOIA. To learn more about the FOIA, please visit FOIA.gov.

Getting better all the time

We're currently in the alpha stage of development, which means our site might not have every feature you're looking for.

Please let us know what's working and what you'd like to see in the future - we're actively improving openFOIA, and we base our improvements on feedback from users like you! Email your suggestions to 18f-foia@gsa.gov, or visit our issue tracker here.

Developers should check out the developer resources page.

Current Features

  • Searchable contact information

    openFOIA consolidates the contact information for all federal agencies’ and components’ FOIA offices. It’s easy to search, and it provides links to each agency’s online FOIA request form (when available), allowing users to start their requests more quickly.

  • Agency description search

    openFOIA lets you search by some keywords, which is helpful if you’re not sure which agency to contact. For example, a search for 'credit cards' brings up the Consumer Financial Protection Bureau, and a search for 'criminal laws' brings up the Bureau of Alcohol, Tobacco, Firearms and Explosives.

  • Acronym search

    Some agencies are better known by their acronyms. openFOIA allows users to search by acronym instead of full agency (or component) title. For example, a search for 'FBI' brings up the Federal Bureau of Investigation.

  • FOIA libraries

    Many agencies provide commonly requested responsive documents in online libraries or reading rooms. openFOIA links to these libraries, letting users see if the information they want is already available.
